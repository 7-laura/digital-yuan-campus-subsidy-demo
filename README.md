# 数字人民币校园专项补助定向支付系统

本项目是课程设计使用的前端 demo，用于模拟“数字人民币校园专项补助/助学金定向支付系统”的页面与后续业务流程。当前版本已完成项目初始化、基础页面布局、测试环境配置、核心数据类型和 mock 数据，不接入真实数字人民币系统、银行接口、支付接口或区块链网络。

## 技术栈

- React
- TypeScript
- Vite
- Vitest

## 运行方式

```bash
npm install
npm run dev
```

## 测试方式

```bash
npm test
```

## 构建方式

```bash
npm run build
```
